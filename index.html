Privacy Policy

1. Data Collection Philosophy

DiscoCinema prioritizes user privacy above all. **We collect zero personally identifiable information and zero media data.**

2. What We Don't Collect

  • No video/audio files - Never uploaded, scanned, or processed
  • No filenames - We don't know what you're watching
  • No user data - No emails, names, or profiles stored
  • No chat history - Session chat is ephemeral
  • No analytics - No tracking pixels or behavioral data

3. Sync Data (Ephemeral)

During active sessions only, we temporarily process:

  • Discord User ID (party identification)
  • Current timestamp (sync position)
  • Playback state (play/pause/seeking)
  • Room ID (session routing)

This data exists only in RAM during your session and is discarded immediately after.

4. Local Storage

Your browser may store:

  • UI preferences (dark mode, etc.)
  • Recent room IDs
  • Local sync settings

This data never leaves your device.

5. Third Parties

DiscoCinema operates within Discord infrastructure:

  • Discord's Privacy Policy applies to user profiles
  • Discord servers relay ephemeral sync packets
  • No third-party trackers or services

6. Security

All sync communication uses:

  • WSS (secure WebSocket)
  • TLS 1.3 encryption
  • Ephemeral-only data processing
  • No persistent databases
